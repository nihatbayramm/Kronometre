# 📊 KPSS Çalışma Takip Paneli (https://nihatbayramm.github.io/Kronometre/)

KPSS’ye hazırlananlar için **şık**, **hızlı** ve **kullanımı kolay** bir çalışma takip paneli.  
Kronometreyle ders kaydet, manuel giriş yap, raporları gör ve grafiklerle ilerlemeni izle.

> 🖤 Dark Mode  
> 💛 Sarı vurgu (odak & premium görünüm)  
> ⚡ Tamamen tarayıcı içinde çalışır (backend yok)

---

## ✨ Özellikler

- ⏱ **Canlı Kronometre**
  - Başlat / Durdur / Sıfırla
  - Çalışırken animasyonlu saat efekti

- 💾 **Ders Kaydetme**
  - Kronometreden direkt kayıt
  - Manuel saat–dakika girişi
  - Kayıtlar otomatik **localStorage**’da saklanır

- 🧹 **Tek Tek Silme**
  - Yanlış kaydı tek tuşla sil
  - İstersen tüm raporu sıfırla

- 📚 **Kayıtlı Çalışmalar Paneli**
  - Ders adı + süre listesi
  - Yan panelde düzenli görünüm

- 📈 **Raporlama**
  - Toplam çalışma süresi
  - En çok çalışılan ders
  - Ders bazlı süre dağılımı

- 📊 **Grafik Görünümü**
  - Chart.js ile dinamik grafik
  - “Raporu Göster” butonuyla açılır
